# EaSync AI module (stubs)

This folder contains minimal AI stubs used by the core during development.

- `include/` - public headers for the local AI module.
- `src/` - minimal implementations returning canned JSON responses.
- `schemas.json` - lightweight JSON schemas for requests/responses.

These files are intentionally small and dependency-free. Replace with
real model runtime or Python/C++ inference bindings as needed.
